RUN & TEST INSTRUCTIONS

1) Create and activate a Python virtual environment.

2) Install dependencies:
   pip install -r requirements.txt

3) Start the server:
   uvicorn app.main:app --reload --port 8000

4) Use the interactive docs at:
   http://127.0.0.1:8000/docs

5) Example curl commands:

- Create doctor:
  curl -X POST "http://127.0.0.1:8000/Doctor/" -H "Content-Type: application/json" -d '{"name":"Dr. Priya","age":38,"gender":"Female","speciality":"Cardiology"}'

- Get doctor:
  curl "http://127.0.0.1:8000/Doctor/1"

- List doctors:
  curl "http://127.0.0.1:8000/Doctors/"

- Update:
  curl -X PUT "http://127.0.0.1:8000/Doctors/1" -H "Content-Type: application/json" -d '{"name":"Dr. Priya R","age":39,"gender":"Female","speciality":"Cardiology"}'

- Soft delete:
  curl -X DELETE "http://127.0.0.1:8000/Doctors/1"

6) To create the Word doc with screenshots (manually):
   - Run requests / open the Swagger UI and take screenshots of:
     * Request payload used
     * Response JSON visible in Swagger / curl terminal output
     * pytest output (if you run tests)
   - Open Microsoft Word, paste images and add captions:
     * "API Create Doctor - request and response"
     * "API GET Doctor - response"
     * "pytest results"
   - Save as `EDR_test_results.docx`.

7) Zipping already included in this archive. To deploy, recommended steps:
   - Use Docker (create Dockerfile) and push to Railway/Fly.io/Render.
   - Set an env var for DATABASE_URL (for production DB).
